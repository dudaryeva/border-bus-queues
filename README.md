# border-bus-queues
Project in Power BI to analyze queues on borders

Data source: https://data.gov.ua/dataset/a8909a08-4da6-49c5-9238-6ab227059c51

<b>Purpose</b>
The goal is to explore and visualize how long buses wait at border checkpoints, how queue sizes change over time, and which checkpoints experience the heaviest traffic.
<li>Visualize how queue lengths and waiting times change across different checkpoints and time periods.
<li>Identify patterns and inefficiencies in cross-border traffic flow.
<li>Promote data transparency and informed decision-making in public logistics management.</li>

<b>Tech Stack:</b>
Power BI · DAX · Open Data (data.gov.ua) · Data Visualization
